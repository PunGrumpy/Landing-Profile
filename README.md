## How to use this project

### 1. Clone this repository and run into project

```bash
git clone https://github.com/PunGrumpy/Landing-Profile.git
cd Landing-Profile
```

### 2. Install dependencies

```bash
yarn install
```

### 3. Start development server

```bash
yarn run dev
```
